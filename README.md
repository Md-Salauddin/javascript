
# Data type
evaluate left to right

```javascript
  let x = 16 + 4 + "Hello"; [output: 20Hello]
  let x = "Hello"+ 16 + 4; [output: Hello164]
```

## Dynamic Data Type 

```javascript
  let x; // undefined
  x = 2; // now x is number
  x = "Hello"; // now x is string
```

## Number

```javascript
  let x; 
  x = 12e5; 
  x = 12e-5; 
```

## Array

```javascript
  const words = ["apple", "banana", "cat"]; 
  console.log(words[0]); [output: apple]
```
